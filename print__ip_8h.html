Specialization of a printing ip function for an integral type. Every byte of the number is printed as a separate part of an ip.

Specialization of a printing ip function for a std::tuple. Only the tuple with the same types can be allowed, for example std::tuple<int, int> is allowed and std::tuple<int, long> is not.

Specialization of a printing ip function for a std::list and std::vector. Every element of std::vector or std::list is considered a separate part of an ip.

Specialization of a printing ip function for a std::string. Can also be called with all the types the std::string can be constructed with. The ip is printed as it is without any changes.
